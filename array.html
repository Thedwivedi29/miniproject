Arrays and Array Functions in Python

An array is a collection of elements of the same data type. Python has a built-in array module that allows you to create arrays.

Here are some common array functions in Python:

  • array.array(typecode[, initializer]) - Creates an array with the specified typecode and optional initializer.
  • array.append(x) - Adds an element to the end of the array.
  • array.extend(iterable) - Adds the elements of an iterable to the end of the array.
  • array.insert(i, x) - Inserts an element at the specified index.
  • array.pop([i]) - Removes and returns the element at the specified index, or the last element if no index is specified.
  • array.remove(x) - Removes the first occurrence of the specified element.
  • array.reverse() - Reverses the order of the elements in the array.
  • array.sort(key=None, reverse=False) - Sorts the elements in the array in ascending order by default, or in descending order if reverse is True.

Examples

Creating an array:

import array
a = array.array('i', [1, 2, 3, 4, 5])

Adding an element to the end of the array:

a.append(6)

Adding the elements of another array to the end of the array:

b = array.array('i', [7, 8, 9])
a.extend(b)

Inserting an element at a specific index:

a.insert(0, 0)

Removing the last element of the array:

a.pop()

Removing the first occurrence of an element:

a.remove(3)

Reversing the order of the elements:

a.reverse()

Sorting the elements in ascending order:

a.sort()

Sorting the elements in descending order:

a.sort(reverse=True)
