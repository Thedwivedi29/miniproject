Dictionaries in Python

Dictionaries are a type of data structure in Python that store key-value pairs. Each key in a dictionary maps to a specific value, allowing you to easily retrieve the value associated with a given key.

functions in Dictionaries

  • dict(): creates a new dictionary
  • len(): returns the number of key-value pairs in the dictionary
  • clear(): removes all key-value pairs from the dictionary
  • copy(): returns a shallow copy of the dictionary
  • get(key, default): returns the value for the specified key, or the default value if the key is not found
  • items(): returns a view object that contains the key-value pairs of the dictionary
  • keys(): returns a view object that contains the keys of the dictionary
  • pop(key, default): removes the specified key-value pair from the dictionary and returns the value, or the default value if the key is not found
  • popitem(): removes and returns the last inserted key-value pair in the dictionary
  • setdefault(key, default): returns the value for the specified key, or sets the key to the default value if the key is not found
  • update(other_dict): adds the key-value pairs from another dictionary to the dictionary
  • values(): returns a view object that contains the values of the dictionary

Here are some examples of using these functions:

Creating a Dictionary

You can create a dictionary in Python by enclosing a set of key-value pairs in curly braces {} separated by commas. Here's an example:

          
            # Creating a dictionary
            my_dict = {'apple': 2, 'banana': 3, 'orange': 4}

Accessing Values in a Dictionary

You can access the values in a dictionary by referencing the corresponding key. Here's an example:

          
            # Accessing a value in a dictionary
            print(my_dict['apple']) # Output: 2

Updating a Dictionary

You can update the values in a dictionary by simply assigning a new value to a specific key. Here's an example:

          
            # Updating a value in a dictionary
            my_dict['apple'] = 5
            print(my_dict) # Output: {'apple': 5, 'banana': 3, 'orange': 4}

Looping Through a Dictionary

You can loop through a dictionary using a for loop to access each key-value pair. Here's an example:

          
            # Looping through a dictionary
            for key, value in my_dict.items():
                print(key, value)

This will output:

          
            apple 5
            banana 3
            orange 4

Some More Examples


            my_dict = dict(name='John', age=30, country='USA')
            print(len(my_dict)) # output: 3
            
            my_dict.clear()
            print(my_dict) # output: {}
            
            my_dict = dict(name='John', age=30, country='USA')
            my_dict_copy = my_dict.copy()
            print(my_dict_copy) # output: {'name': 'John', 'age': 30, 'country': 'USA'}
            
            print(my_dict.get('name', 'Unknown')) # output: John
            print(my_dict.get('address', 'Unknown')) # output: Unknown
            
            print(my_dict.items()) # output: dict_items([('name', 'John'), ('age', 30), ('country', 'USA')])
            
            print(my_dict.keys()) # output: dict_keys(['name', 'age', 'country'])
            
            print(my_dict.pop('age', 25)) # output: 30
            print(my_dict) # output: {'name': 'John', 'country': 'USA'}
            
            print(my_dict.popitem()) # output: ('country', 'USA')
            
            print(my_dict.setdefault('address', 'Unknown')) # output: Unknown
            print(my_dict) # output: {'name': 'John', 'address': 'Unknown'}
            
            my_dict.update({'city': 'New York'})
            print(my_dict) # output: {'name': 'John', 'address': 'Unknown', 'city': 'New York'}
            
            print(my_dict.values()) # output: dict_values(['John', 'Unknown', 'New York'])
