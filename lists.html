Python Lists and Functions

Python lists are a data structure that allow you to store multiple values in a single variable. Lists are created using square brackets [] and each item in the list is separated by a comma.

Here are some common list functions:

  • len(list): returns the number of items in the list
  • list.append(item): adds an item to the end of the list
  • list.insert(index, item): inserts an item at a specific index
  • list.remove(item): removes the first occurrence of an item from the list
  • list.pop(index): removes and returns the item at a specific index
  • list.sort(): sorts the items in the list in ascending order
  • list.reverse(): reverses the order of the items in the list

Examples

Here are some examples of using lists and their functions:

    
      # Creating a list
      my_list = [1, 2, 3, 4, 5]
      
      # Finding the length of the list
      length = len(my_list)
      
      # Adding an item to the end of the list
      my_list.append(6)
      
      # Inserting an item at a specific index
      my_list.insert(2, 10)
      
      # Removing an item from the list
      my_list.remove(3)
      
      # Removing and returning an item at a specific index
      popped_item = my_list.pop(1)
      
      # Sorting the items in the list
      my_list.sort()
      
      # Reversing the order of the items in the list
      my_list.reverse()
