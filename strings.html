Strings and String Functions in Python

Strings in Python

Strings are a sequence of characters in Python. They can be enclosed in single quotes ('...') or double quotes ("...").

Here's an example:


            # Declare a string variable
            my_string = "Hello, World!"
            
            # Print the string
            print(my_string)
            
            # Accessing characters in a string
            print(my_string[0])    # Output: 'H'
            print(my_string[6])    # Output: 'W'
            print(my_string[-1])   # Output: '!'

The output of the above code would be:


            Hello, World!
            H
            W
            !

Python provides a number of built-in functions for working with strings. Here are some of them:

  • len(): returns the length of a string
  • str.lower(): returns a string in lowercase
  • str.upper(): returns a string in uppercase
  • str.replace(old, new): replaces all occurrences of old with new in a string
  • str.strip(): removes leading and trailing whitespace from a string
  • str.split(): splits a string into a list of substrings based on a delimiter

Here's an example:


            # Declare a string variable
            my_string = "Hello, World!"
            
            # Using built-in functions
            print(len(my_string))                 # Output: 13
            print(my_string.lower())              # Output: hello, world!
            print(my_string.upper())              # Output: HELLO, WORLD!
            print(my_string.replace("World", "Python"))    # Output: Hello, Python!
            print(my_string.strip())              # Output: Hello, World!
            
            # Splitting a string
            my_string = "Hello,Python,World!"
            my_list = my_string.split(",")
            print(my_list)                        # Output: ['Hello', 'Python', 'World!']

The output of the above code would be:


            13
            hello, world!
            HELLO, WORLD!
            Hello, Python!
            Hello, World!
            ['Hello', 'Python', 'World!']

String Functions

len()

Returns the length of a string.

str = "Hello, world!" print(len(str))

Output: 13

lower()

Returns a copy of the string in lowercase.

str = "Hello, world!" print(str.lower())

Output: hello, world!

upper()

Returns a copy of the string in uppercase.

str = "Hello, world!" print(str.upper())

Output: HELLO, WORLD!

strip()

Returns a copy of the string with leading and trailing whitespace removed.

str = " Hello, world! " print(str.strip())

Output: Hello, world!

replace()

Returns a copy of the string with all occurrences of a substring replaced with another substring.

str = "Hello, world!" print(str.replace("Hello", "Hi"))

Output: Hi, world!

split()

Returns a list of substrings separated by a delimiter.

str = "Hello, world!" print(str.split(", "))

Output: ['Hello', 'world!']

join()

Joins a list of strings with a specified separator.

mylist = ['apple', 'banana', 'cherry'] separator = " and " print(separator.join(mylist))

Output: apple and banana and cherry
