Data Types in Python

Python has several built-in data types, which can be classified as follows:

  • Numeric
  • Sequence
  • Boolean
  • Set
  • Dictionary

Numeric Data Types

Numeric data types in Python are used to represent numeric values. There are three types of numeric data types in Python:

  • Integer (int): represents positive or negative whole numbers.
  • Float (float): represents real numbers with decimal points.
  • Complex (complex): represents complex numbers with a real and imaginary component.

Examples:

		x = 5 # integer
		y = 2.7 # float
		z = 2 + 3j # complex

Sequence Data Types

Sequence data types in Python are used to represent sequences of values. There are three types of sequence data types in Python:

  • String (str): represents a sequence of characters.
  • List (list): represents a sequence of values, which can be of different data types.
  • Tuple (tuple): represents a sequence of values, which can be of different data types, but are immutable (cannot be changed).

Examples:

		name = "John" # string
		numbers = [1, 2, 3] # list
		values = (4, "hello", 2.7) # tuple

Boolean Data Type

The Boolean data type in Python is used to represent truth values, which can be either True or False.

Example:

		x = True # boolean
		y = False # boolean

Set Data Type

The Set data type in Python is used to represent a collection of unique values, which can be of different data types.

Example:

		set1 = {1, 2, 3} # set
		set2 = {"apple", "banana", "orange"} # set

Dictionary Data Type

The Dictionary data type in Python is used to represent a collection of key-value pairs, where each key is unique and associated with a value.

Example:

		person = {"name": "John", "age": 30, "city": "New York"} # dictionary
