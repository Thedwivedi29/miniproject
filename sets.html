Sets in Python

In Python, a set is an unordered collection of unique elements. The set data type is mutable and iterable. It is defined by enclosing a comma-separated list of elements in curly braces {}.

Creating a Set

A set can be created in two ways:

  • By using curly braces {}
  • By using the set() constructor
# Creating a set using curly braces
          fruits = {'apple', 'banana', 'orange'}
          
          # Creating a set using set() constructor
          numbers = set([1, 2, 3, 4, 5])

Set Functions

Python provides a number of built-in functions for sets. Some of the commonly used functions are:

  • union(): returns the union of two sets
  • intersection(): returns the intersection of two sets
  • difference(): returns the difference of two sets
  • set(): This creates a new empty set.
  • set(iterable): This creates a new set and adds the elements of the iterable to it. The iterable can be any sequence type (e.g., list, tuple, string).
  • .add(element): This adds an element to a set. If the element is already in the set, nothing happens.
  • .remove(element): This removes an element from a set. If the element is not in the set, it raises a KeyError.
  • .discard(element): This removes an element from a set if it is present. If the element is not in the set, nothing happens.
  • .pop(): This removes and returns an arbitrary element from a set. If the set is empty, it raises a KeyError.
  • .clear(): This removes all elements from a set.
  • len(set): This returns the number of elements in a set.
  • in: This returns True if an element is in a set, and False otherwise.
  • |: This returns the union of two sets.
  • &: This returns the intersection of two sets.
  • -: This returns the difference between two sets (i.e., the elements that are in one set but not the other).
  • ^: This returns the symmetric difference between two sets (i.e., the elements that are in one set or the other, but not both).

Here are some examples of how you might use these functions:

# Creating two sets
          set1 = {1, 2, 3}
          set2 = {3, 4, 5}
          
          # Union of set1 and set2
          print(set1.union(set2)) # Output: {1, 2, 3, 4, 5}
          
          # Intersection of set1 and set2
          print(set1.intersection(set2)) # Output: {3}
          
          # Difference of set1 and set2
          print(set1.difference(set2)) # Output: {1, 2}
          
          set3 = set1 | set2  # union
          set4 = set1 & set2  # intersection
          set5 = set1 - set2  # difference
          set6 = set1 ^ set2  # symmetric difference
          
          print(set3)  # {1, 2, 3, 4}
          print(set4)  # {2, 3}
          print(set5)  # {1}
          print(set6)  # {1, 4}
