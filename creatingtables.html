Creating a Table in MySQL

To store data in a MySQL database, you'll need to create one or more tables to organize the information. Here are the steps to create a table:

  1. Open the MySQL command-line interface by entering the following command in your terminal or command prompt: mysql -u root -p
  2. Enter your MySQL root password when prompted.
  3. Select the database you want to use by entering the following command: USE database_name;
  4. Create a new table by entering the following command, replacing "table_name" with the name you want to give your new table: CREATE TABLE table_name (column1_name data_type1, column2_name data_type2, ...);
  5. Replace "column1_name" and "data_type1" with the name and data type for your first column, and repeat for each additional column you want to add.


  6. Confirm that the table was created by entering the following command: SHOW TABLES;




  7. Your new table should appear in the list of tables.



Code Example


    mysql -u root -p
    Enter your MySQL root password when prompted.
    USE database_name;
    CREATE TABLE company (
      id INT NOT NULL AUTO_INCREMENT,
      name VARCHAR(255) NOT NULL,
      address VARCHAR(255) NOT NULL,
      PRIMARY KEY (id)
    );
    SHOW TABLES;

Showing a Table in MySQL

To view the data in a MySQL table, you can use the SELECT statement. Here's an example of how to select all data from a table named "company" and display the results:


    SELECT * FROM company;

This will display all the data in the "company" table, including the ID, name, and address columns.
